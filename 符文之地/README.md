# 符文之地 (Rune Land)

> 超大型Roguelike ARPG — 4职业×50层地下城×程序化生成

## 游戏设计目标

### 职业系统
| 职业 | 定位 | 核心属性 | 技能类型 |
|------|------|----------|----------|
| 战士 | 高攻物理 | 力量、体质 | 斩击、盾击、战吼 |
| 法师 | 元素魔法 | 智力、意志 | 火球、冰霜、雷电 |
| 游侠 | 敏捷暴击 | 敏捷、幸运 | 疾风箭、多重射击、陷阱 |
| 圣职 | 治疗辅助 | 感知、意志 | 治疗、神圣制裁、祝福 |

### 地下城系统
- **5个地区**：阴暗森林 → 荒芜沙漠 → 冰霜要塞 → 烈焰地狱 → 虚空神殿
- **每地区10层**：共50层程序化生成地下城
- **敌人数量**：75种普通敌人 + 30种首领

### 物品系统 (120种)
- 武器：剑、斧、锤、弓、法杖
- 护甲：头盔、胸甲、手套、护腿、靴子
- 药水：生命、魔法、体力、速度
- 卷轴：火球、冰霜、治疗、传送
- 戒指/项链：属性加成、特殊效果

### 技能系统 (50种)
- 每职业12+种专属技能
- 主动技能：消耗MP释放
- 被动技能：永久生效

### 经济系统
- 商店：购买基础装备
- 锻造：升级装备品质
- 炼金：制作药水
- 附魔：为装备附加魔法属性

### 任务系统
- 30+任务链
- 主线+支线任务
- 随机事件

### 成就系统
- 50+成就解锁
- 进度追踪

### 战斗系统
- 连击系统：连续命中累积连击数
- 暴击系统：暴击率×暴击伤害
- 元素反应：火+冰=蒸发、电+水=导电

## 技术栈

```
HTML5 Canvas + 原生JavaScript + JSON
无外部依赖
```
